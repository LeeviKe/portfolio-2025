Movie score quiz

A simple frontend app for guessing between 2 movies, which one has the higher viewer score on TMDB.

Features:

The app gets the movie data from different TMDB Apis chosen with a randomizer. The game process consists of getting 2 movies, player choosing one, checking if the chosen movie has the higher score and either continuing with new movies or ending the game and showing the gotten score. The app only has one route, and is completely build with using modals.

Techs:

Svelte, JavaScript

What I learned:

This is my first completed frontend app ever. Even though it doesn't have anything fancy and with the current skills I could make it 100 times better, it still turned out pretty well and I wanted to showcase it here. I learned much about everything releated to frontend development, such as Svelte, api usage, game logic and modals.
