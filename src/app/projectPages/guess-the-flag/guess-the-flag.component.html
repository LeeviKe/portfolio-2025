Guess the flag

A simple frontend mobile app I made in the spring of 2025. In the game, the user is shown a random flag and they have limited time to try and choose the right one from 4 options.

Features:

This is a hybrid mobile app, made with Ionic. The game has different levels on time difficulty and flag regions that the player can alter for their liking. The code gets the flags form a public api and uses a randomizer for getting the flags for each round. Best scores for each game settings are being stored in the device with Ionic storage. Overall a simple mobile app with some decent game logic inside of it.

Techs:

Ionic, Angular, TypeScript.

What I learned:

This was my first complete mobile app project. I learned a lot about mobile app development, Ionic + storage, emulator usage and app packing into an apk. I also got to see how Angular can be used in a different way from the classic web development.
